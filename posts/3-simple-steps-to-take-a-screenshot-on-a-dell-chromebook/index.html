First, press the “Ctrl” key and the “Override” key at the same time. On a full-size keyboard, the “Override” key is the same as the F5 key, while on a smaller Chromebook, it is usually located near the top row of the keys. Once the keys are pressed, the screen will appear to darken.

Second, press the “Ctrl” and “Window Switcher” keys at the same time. This will take a full-screen screenshot and save it to your Chrome OS files.

Last, open Chrome OS Files and select the “Downloads” folder. You should see the screenshot saved there. You can then edit it using an app like Paint or Picasa, share it, or whatever you need to do with it.
